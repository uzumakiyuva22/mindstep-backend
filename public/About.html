How MindStep Works End-to-End

1. Structured lesson path

Students move through ordered lessons per course. Progress is tracked by lesson and task level.

2. Project checkpoints

Project lessons require practical file submissions (ZIP/DOC/PDF based on lesson rules).

3. Admin review gate

Submission status is Pending until admin approval. Lesson completion is blocked until approval.

4. Completion integrity

Course reaches true completion only when all required lessons and project approvals are done.

5. Certificate unlock

After rules are satisfied, certificate claim is enabled with unique ID and downloadable document.

Frequently Asked Questions

Can I complete project lessons without admin review?

No. For project lessons, admin approval is required before the lesson is marked complete.

When will certificate button appear?

After all required lessons are complete and all project submissions for the course are approved.

Can rejected submission be resubmitted?

Yes. Rejected submissions are reopened for re-upload with feedback from admin review.
